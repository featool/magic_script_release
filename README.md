## 特性
一个可视化OpenSeespy模型的软件。可直接运行Openseespy脚本，获得3D模型。

## roadmap
- v0.0.1
   - 搭建基础框架
   - 实现简单脚本的执行
- v0.0.2
   - 增强3D模型视图交互能力
   - 增强模型树功能
   - 实现模型的边界条件、载荷的渲染
- v0.0.3
   - 支持Tcl/TK模型
   - 梁截面的实体渲染
   - 纤维束截面的渲染
- v0.0.4
   - 后处理结果的提取与渲染

## 关于更新
  有新版本会弹出Toast提示，但不会自动更新，需用户手动下载安装。主要原因是MagicScript的发版使用github，众所周知的原因自动更新大部分情况下会失败。
