安装与使用

安装

目前只提供windows安装包。双击安装程序即可安装。

运行脚本

提供openseespy脚本文件打开、保存、执行三个按钮。用户自己的openseespy脚本不用做任何修改，即使带有matplotlib绘图命令也可直接运行无需修改。但带绘图的代码会卡住，等python代码执行结束之后才能看到模型本身。
